# ctfd-enum

**ctfd-enum** is an enumeration tool for [CTFd](https://ctfd.io) instances. It supports enumeration of registered users and emails, whitelisted email domains, registration codes, and brute-force testing of login with optional IP rotation to mitigate rate limiting.

> [!WARNING]
> The project is under development.
